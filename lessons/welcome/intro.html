Complete Intro to SQLite

Watch on Frontend Masters

Hello! And welcome to the Complete Intro to SQLite, as taught by Brian Holt for Frontend Masters. In this course you will get learn the basics of SQL, the core features of SQLite, how to build a Node.js app with SQLite, what interesting and unique to SQLite as compared to other databases, some advance JSON ways to use SQLite, and what the future holds for SQLite with things like Litestream, LiteFS, and libSQL.

Who is this course for?

You, hopefully. This course shouldn't be your first exposure to programming or the command line, and it does use a bit of JavaScript so it would help if you knew bit of JavaScript. There's also some very light Docker use but I'll explain all of it so it's okay if you don't need Docker. Luckily I've taught courses on these subjects if you want to brush up!

  • Complete Intro to Web Dev
  • Complete Intro to Linux and the CLI
  • Complete Intro to Containers

These are great if you need help on any of those subjects. You do not any exposure to SQL prior to this course, the first part of it is basic querying in SQL. It is a brief intro, and if you want SQL, you can take my Complete Intro to SQL which uses Postgres but nearly all of it would apply to SQLite as well.

Where to File Issues

I write these courses and take care to avoid making mistakes. However, when teaching hours of material, mistakes are inevitable, both here in the grammar and in the course with the material. However, I (and the wonderful team at Frontend Masters) are constantly correcting the mistakes so that those of you that come later get the best product possible. If you find an error, we'd love to fix it. The best way to do this is to open a pull request or file an issue on the GitHub repo. While I'm always happy to chat and give advice on social media, I can't be tech support for everyone. And if you file it on GitHub, those who come later can Google the same answer you got.
